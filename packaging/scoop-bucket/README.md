# scoop-bucket

Scoop bucket for [cairn](https://github.com/R0kshan/cairn).

```powershell
scoop bucket add cairn https://github.com/R0kshan/scoop-bucket
scoop install cairn
```

> `bucket/cairn.json` is generated and pushed automatically by cairn's release workflow on every `v*` tag — do not edit it by hand.
